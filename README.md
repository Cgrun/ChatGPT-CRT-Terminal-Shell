# ChatGPT-CRT-Terminal-Shell
Oracle CRT Terminal Shell is a Tampermonkey userscript that transforms ChatGPT into a glowing retro CRT terminal with curved WebGL rendering, terminal-style input, popup composer controls, progress bar, preserved code formatting, and one-click code copying.
